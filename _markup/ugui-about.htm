About Universal GUI

The Universal Graphical User Interface (UGUI) allows you to create your own desktop applications in minutes.
UGUI uses NW.js, jQuery, Bootstrap, Bootswatch, Sass, DropZone Plugin, Bootstrap Slider, strip-json-comments, nw-contextmenu.
NW.js uses Node.js/IO.js, parts of Chromium, Webkit, and V8. For a full listing of technologies used by NW.js visit nwjs.io.
