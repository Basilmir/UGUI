About Universal GUI

The Universal Graphical User Interface (UGUI) lets you to create your own desktop applications in minutes.

UGUI uses NW.js, Chromium, jQuery, Bootstrap, Bootswatch, Sass, ezdz, Bootstrap Slider, Bootstrap Dropdowns Enhancement, strip-json-comments, nw-contextmenu.

NW.js uses Node.js/IO.js, parts of Chromium, Webkit, and V8. For a full listing of technologies used by NW.js visit nwjs.io.
